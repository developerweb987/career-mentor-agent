# 💼 Career Mentor Agent

A Python-based multi-agent system that guides students in exploring careers, learning required skills, and discovering real-world job roles — built using modular agents and tools.

---

## 🧠 What It Does

This project simulates a smart career counseling system using multiple agents:

- **CareerAgent** – Suggests career fields based on student interests
- **SkillAgent** – Provides a skill-building roadmap for the chosen career
- **JobAgent** – Shows real-world job roles relevant to that career

---

## ⚙️ Technologies Used

- Python 3
- Modular architecture with multiple agents
- Custom Tool: `get_career_roadmap()`
- CLI-based interaction
- Git version control
